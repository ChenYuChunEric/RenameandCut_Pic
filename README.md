# 新生照片批次處理與更名工具

本工具專為小學一年級新生入學照片處理設計。行政人員可透過簡單的圖形化介面，將家長或學生上傳的照片檔名（身分證字號）批次更名為學號，並同步進行規格轉檔、等比例縮放裁切，更提供**即時雙視窗預覽與裁切框微調**，解決照片人頭偏上、偏下或尺寸不一的問題。

---

## 🌟 核心功能

1. **批次更名 (Excel 對照)**：支援讀取 `.xlsx` 或 `.xls` 對照表，自定義選取「身分證字號」與「學號」的欄位標頭，自動將符合身分證字號的照片檔名更換為學號。
2. **規格轉檔**：支援將任何常見格式（JPG、PNG、BMP、WEBP、TIFF）圖片，統一批次轉檔輸出為指定的 `.png` 或 `.jpg` (JPEG) 格式。
3. **等比例裁切與縮放**：
   - 內建常用規格：**2 吋證件照** (354 x 472 像素)、**1 吋證件照** (283 x 378 像素)。
   - 支援自訂任意寬高尺寸。
   - 提供「自動等比例縮放 + 滿版裁切」，避免照片變形或留黑邊。
4. **即時雙視窗預覽與微調**：
   - **左側預覽**：顯示原始照片並疊加紅色裁切外框指示線。
   - **右側預覽**：即時呈現裁切縮放後的實際效果。
   - **位移/縮放滑桿**：可即時調整照片的「垂直位移（上下移動）」與「裁切比例（放大人像）」，確保學生的頭部位置完美對齊。
   - **套用功能**：可單張微調，亦可一鍵「套用至全部照片」，省時省力。
5. **安全防護**：非同步線程（Threading）執行批次處理，軟體不卡死；遇到無對照學號的照片會以「原名 + 未對照」另存輸出，不遺失任何檔案。

## 🛠️ 開發環境與運行

本專案使用 **Python 3.13** 開發。

### 1. 建立虛擬環境與安裝依賴套件

若您在開發環境中，請於本專案目錄下執行：

```powershell
# 建立虛擬環境
python -m venv venv

# 啟用虛擬環境 (Windows PowerShell)
.\venv\Scripts\activate

# 安裝依賴套件
pip install -r requirements.txt
```

### 2. 執行程式

在虛擬環境啟用狀態下，執行主程式：

```powershell
python app.py
```

---

## 📦 打包成獨立 EXE 執行檔 (免安裝 Python)

若要提供給學校行政人員雙擊直接使用，可以使用 **PyInstaller** 將程式打包成獨立的 `.exe` 執行檔。

### 打包步驟

在專案目錄下執行以下指令：

```powershell
# 打包為單一 EXE 檔案（無 Console 背景視窗，帶有現代化 GUI）
.\venv\Scripts\pyinstaller --noconsole --onefile --name="新生照片批次處理工具" app.py
```

打包完成後，您可以在生成的 `dist/` 資料夾中找到 `新生照片批次處理工具.exe`。

## 📖 操作指南

1. **載入對照表**：點擊左側「1. Excel 對照表設定」的 **選擇 Excel 檔案** 按鈕，載入新生名冊。
   - 載入後，系統會智慧預測並設定「身分證欄位」與「學號對應欄位」。若預測不準確，可透過下拉選單手動選擇對應的欄位標頭。
2. **選擇照片來源**：點擊「2. 照片來源路徑」的 **選擇照片資料夾**，選擇家長上傳的原始照片目錄。
   - 載入後，右側上方的清單會列出所有照片與比對到的學號，狀態會顯示「已對照」、「無對照學號」或「待處理」。
3. **設定輸出規格**：
   - 選擇輸出尺寸（如 2 吋 354x472 px）。
   - 設定輸出格式（如 JPG）。
   - 選擇或保留預設的輸出資料夾路徑。
4. **預覽與裁切微調**：
   - 點擊右側上方照片清單中的任一張照片，下方即時顯示「原圖+裁切框」與「處理後預覽」。
   - 若人像位置不佳，可拉動「**垂直位移**」與「**裁切框大小**」滑桿進行微調。
   - 點擊 **置中對齊** 或 **偏上對齊** 可快速復位。
   - 若照片拍攝比例均相似，調整好一張後點擊 **將此裁切設定套用至全部照片**。
5. **開始處理**：點擊 **開始批次處理**。進度條跑完後，系統將跳出完成提示，即可在輸出資料夾中查看完美更名與裁切後的學生照片！
